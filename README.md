# ESLint Essentials

## Whats included?

- Eslint 8
- Standard config base;
- React plugin;
- React native plugin;
- React Hooks plugin;
- JSX a11y plugin;
- Prettier;

## Setup

### Node (with Typescript)

Install dependencies:

```cmd
npm i -D eslint @eslint-essentials/eslint-config-node-ts
```

Inside `.eslintrc`

```json
{
  "extends": "@eslint-essentials/eslint-config-node-ts"
}
```
