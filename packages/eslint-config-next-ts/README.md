# ESLint Essentials

## Whats included?

- Eslint 8
- Standard config base;
- Next plugin;
- Prettier;

## Setup

### Next (with Typescript)

Install dependencies:

```cmd
npm i -D eslint @eslint-essentials/eslint-config-next-ts
```

Inside `.eslintrc.json`

```json
{
  "extends": "@eslint-essentials/eslint-config-next-ts"
}
```
