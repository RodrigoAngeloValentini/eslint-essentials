## Setup

### React Native (with Typescript)

Install dependencies:

```
npm i -D eslint @eslint-essentials/eslint-config-react-native-ts
```

Inside `.eslintrc.json`

```
{
  "extends": "@eslint-essentials/eslint-config-react-native-ts"
}
```
